システム要件

サーバー

OS非依存

  • Apache2
  • PHP 5.2.4 以降
  • MySQL5.1 以降

クライアント

推奨ブラウザ

  • IE8以上
  • Firefox3.6以上
  • GoogleChrome10以上

インストール手順

1.ダウンロードしてきたSetucoCMS_1.5.2.zipを解凍します

2.[SetucoCMS_1.5.2/public]ディレクトリ内にある.htaccess.sampleをコピーします

内容の変更は基本的には不要です。
(FTPにアップ後、名前を.htaccessに変更します)

3.解凍してできたSetucoCMS_1.5.2の中身をFTPソフトを使ってアップロードします

4.ステップ2で作った[.htaccessのコピー]の名前を.htaccessにします

5.ディレクトリの属性を変更します

以下の4つのディレクトリの属性を777にします。

  • /public/
  • /public/media/
  • /application/
  • /application/configs/

6.公開ディレクトリの設定

使用しているサーバーなどによって、多少設定方法が異なりますが、ご自分の環境にあてはめて進めてください。

ステップ3の、FTPでアップロードしたフォルダの中にあるpublicディレクトリにアクセスした際のURLが、http://example.com/になるように設定します。

独自ドメインを取得されている方

レンタルサーバーの管理画面に独自ドメインを設定するメニューがあるので、そちらに移動します。

独自ドメインの設定項目の中に、公開フォルダ設定という設定項目がありますので、サーバーのルートディレクトリから、ステップ3でアップしたpublicディレクトリまでのパスを入力します。

(例)
/home/users/2/co.jp-testserver/web/public

画像1-1:レンタルサーバー「ロリポップ」の説明(クリックで拡大します)

画像1-2:レンタルサーバー「ヘテミル」の説明(クリックで拡大します)

設定が完了したら、独自ドメインの後に/installを付けたURLにアクセスします。
するとブラウザ上にインストール画面が表示されます。

(例)
http://独自ドメイン名/install

設定が反映されるまでに1時間ほどかかる場合がありますので、表示されない場合はしばらくたってから再度お試しください。

独自ドメインを持っていない方

独自ドメインを持っていない方は、アップロードしたディレクトリにpublic/installをつけてサイトにアクセスするとインストール画面が表示されます。

http://サイトのURL/public/install

VirtualHostの設定をする方

VirtualHostの設定を参考に設定します。

7.データエースの作成

SetucoCMSをインストールするデータベースを作成します。
SetucoCMSでの推奨環境はMySQLですので、データベースを作成する場合はphpmyadminにアクセスします。

レンラルサーバーを利用している場合は、レンタルサーバーの管理画面にデータベースを使用するなどのメニューがありますので、そちらからphpmyadminにアクセスしてインストール用のデータベースを作成します。

8.インストール

インストール画面にアクセスして、セットアップに進む =>をクリックすると、インストールに必要な情報の入力が求められます。
これらをすべて入力後、設定を確認するボタンを押し先に進みます。

注意
[必須] データベースサーバーホストがデフォルトではlocalhostになっていますが、レンタルサーバーによっては変更が必要な場合がありますのでご確認ください。

入力したデータに問題がなければ画像2-3のように入力情報確認画面が表示されます。
問題に不備がある場合は画像2-4のようにエラーが表示されます。

この場合は入力情報を確認してください。

入力情報確認画面の情報に問題がなければ、送信を押してください。
設定完了！画面が表示されればインストールは成功です。

閲覧側、管理側画面に移動してサイト制作をお楽しみください。

VirtualHostの設定

インストール時に必要であればVirtualHostの設定を行います。
設定ファイルに以下を追加します。


<VirtualHost *:80>
	 DocumentRoot "/path/to/SetucoCMS/public"
	 ServerName localhost
	 <Directory "/path/to/SetucoCMS/public">
		 Options Indexes MultiViews FollowSymLinks
		 AllowOverride All
		 Order allow,deny
		 Allow from all
	 </Directory>
</VirtualHost>

.htaccessの設定

インストールの手順の際、.htaccessは編集不要と説明しましたが、VirtualHotを設定した場合には編集する必要があります。

  1. SetucoCMS_1.5.2/publicディレクトリ内にある.htaccess.sampleをコピーします
  2. コピーしてできた.htaccess.sampleのコピーを.htaccessに名前の変更します
  3. .htaccessの中に#RewriteBase /setucocmsという記述がありますので、先頭の[#]を外します。
  4. /setucocmsのパスを、VirtualHostの設定で指定したDocumentRootのパスと同じにします。

以上で終了です。
